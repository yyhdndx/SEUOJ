# 当前功能总览

本文档用于快速说明当前 SEU OJ 已经实现的功能边界，适合组内协作、演示准备和任务分工。

## 1. 用户与权限

- 用户注册、登录、JWT 鉴权
- 当前用户信息查询
- 个人资料修改、密码修改
- 角色区分：`student`、`teacher`、`admin`

## 2. 题库与题解

- 题目列表、详情、搜索
- 题目测试点管理
- 管理员题目 CRUD
- 题目统计接口
- 题解列表与教师/管理员题解管理

## 3. 提交与判题

- 多语言提交：C / C++ / Python / Java / Go / Rust
- `Run` 样例测试
- 正式 `Submit`
- 我的提交列表与详情
- Docker 沙箱判题
- 独立 Judge Worker
- Redis 判题队列
- 日志落盘

## 4. 比赛模块

- 比赛列表、详情、报名
- 比赛题目页
- 比赛榜单
- 榜单封榜 / 冻结
- 赛后 practice
- 比赛公告
- 管理员比赛 CRUD

## 5. 教学模块

- 题单（playlist）
- 班级（classroom）
- 作业 / 考试（assignment）
- 学生加入班级
- 学生查看作业、完成练习
- 教师管理题单、班级、作业
- 教师班级分析、作业完成分析
- 教师端搜索 / 状态筛选

## 6. 社区与内容

- 站内公告
- 排行榜
- 论坛发帖 / 回帖
- 帖子置顶 / 锁帖

## 7. 管理后台

- 用户管理
- 提交管理与重判
- 比赛管理
- 公告管理
- 题目管理
- 统计接口

## 8. 前端现状

- 静态 SPA
- 已按业务域拆分 JS / CSS
- 题目页采用左右工作台布局
- 教学、比赛、论坛、管理页已分模块维护

## 9. 当前最适合继续由组员完善的方向

- 比赛页体验与回归检查
- 论坛分区、搜索与分页
- 教学模块展示优化
- 管理页筛选与统计增强
- 文档与演示材料补全
